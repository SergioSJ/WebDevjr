Back End Developer

Un Back End Developer es un programador que se encarga de desarrollar y programar el codigo para contenido de una pagina web desde la parte del servidor y base de datos para que las acciones y solicitudes de un usuario se lleven a cabo de manera eficiente y segura, No es obligatorio que un mismo desarrollador se encargue de realizar Front End, pero si es importante y necesario que tenga al menos conocimientos básicos de lo que se realiza en el Front end

Los lenguajes de programación mas comunes para desarrollar back End son:

  • Java
  • C#
  • Python
  • Java Script
  • PHP
  • C

Las bases de datos que se utilizan para el Back End son:

  • SQL
  • Oracle
  • NoSQL

Servidores mas utilizados para derarrollo Back End

  • Amazon Web Services
  • Azure (Microsoft)
  • Google Cloud Patform
