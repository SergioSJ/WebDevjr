Front end Developer

El desarrollador de Frot End es el que se encarga de diseñar y crear el contenido visual de un sitio web, que es lo que un usuario puede visualizar
en su pantalla cada que accede a una pagina de internet/sitio web.El desarrollador Front End tambien debe contar con conocimientos básicos en
Full Stack y Banck End Developers para poder condinarse en cuanto a las necesidades del cliente para poder crear un sitio web que cumpla con todas
las espectativas y necesidades de este

Los principales lenguajes de programación que necesitas aprender para ser Front End Developer son:

  1. HTML 5
  2. CSS 3
  3. JS: Java Script
  4. Node JS
  5. React
  6. Angular

Algunas de las herramientas que necesitaras para comenzar a poner en practica los ejercicios que aprendas son:

  • Editor de texto de tu preferencia (Sublime, Atom, VS Code, etc)
  • Navegador de internet de tu preferencia
  • Controlador de versiones (Git, GitHub)
