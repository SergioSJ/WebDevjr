Full Stack Developer

El desarrollador full stack es aquel que se tiene conocimientos de varios lenguajes de programación que le permiten contar con un grupo de
habilidades (stack) que le ayudan a poder desempeñar labores tanto del Front End como en el Back End, lo cual lo comvierten en un desarrollador más
completo que le permiten desempeñar una mayor cantidad de tareas

Los principales lenguajes de programación y herramientas que mejor complementan a un desarrollador Full Stack son :

  • Linux
  • MySQL
  • PHP
  • Angular
  • Java script
  • Phyton
